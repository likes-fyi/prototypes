likes.fyi prototypes
================================================================================

A collection of prototypes for [likes.fyi]  preserved here for historical
purposes, mostly so I have somewhere to link to in my blog posts.

This repo is based on the [miniblog] starter template for [Astro] and
[Tailwind].

[likes.fyi]: https://likes.fyi
[miniblog]: https://github.com/nicholasdly/miniblog
[Astro]: https://astro.build
[Tailwind]: https://tailwindcss.com/
