[likes.fyi] is an open source toolkit for sharing the links you like with the
people you love. To learn more head to [likes.fyi] or check out the code on
[GitHub][gh-org] or [tangled][tangled-org]. I also keep a [devlog] and you can follow the project on
Bluesky at [@likes.fyi].
